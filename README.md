<h2>Dependencies:</h2>

<h3>Requests Module</h3>

<br>

<h2>How to use:</h2>

<h3>Specific Site: </h3>
python3 HTTPAnalyzer.py -u www.yoursite.com.br


<h3>Search for a domain: </h3>
python3 HTTPAnalyzer.py -uL yourdomain.com.br
